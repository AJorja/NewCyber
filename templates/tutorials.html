Challenge Tutorials

Below are helpful hints and strategies to get you started on each challenge.

Open Sesame

- Start by examining the script’s permissions. Can you run it? If not, think about how to change that.
- When executing, watch for error messages—they often hint at missing dependencies.
- Consider how to install or simulate missing services like MySQL.
- The script might have syntax or logic issues; review it carefully and test small parts.
- Remember, the flag could be hidden inside the script or output files after it runs.

Traffic Jammed

- Use your browser’s developer tools to inspect page elements and source code.
- Look carefully for any hidden comments, scripts, or files that might contain credentials.
- Once you find credentials, SSH into the container to explore the environment.
- Search for files or directories that could hold the flag.
- The flag may be encoded (e.g., base64), so be ready to decode common encodings.
- Remember to check file permissions and think like an attacker trying to find hidden info.

Turbine Takeover

- Begin by extracting the zip file and exploring its folder structure.
- Pay attention to file names and sizes—some files might be distractions.
- Look for clues that indicate which files contain valuable information.
- Use common decryption or decoding techniques on suspicious files.
- Organize your findings logically to piece together how the turbine is controlled.
- Persistence and thoroughness are key—don’t overlook small details.
